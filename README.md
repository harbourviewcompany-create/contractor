# Northcrest Roofing Co. Contractor Lead Capture Site

Static Netlify-ready roofing lead-capture website for `harbourviewcompany-create/contractor`.

## Deploy

Netlify settings:

- Build command: `npm run build`
- Publish directory: `dist`
- Base directory: blank

## Form

The lead form is named `estimate-request` and uses Netlify Forms with a honeypot field. Successful submissions redirect to `/thank-you`.

## Notes

This is a demo contractor site. Replace contact details, service area, licensing, insurance, warranty and review claims before adapting it for a real roofing company.
